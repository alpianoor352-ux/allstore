Kebijakan Privasi

ALL STORE berkomitmen untuk menjaga keamanan dan kerahasiaan data pelanggan.

1. Data yang Dikumpulkan

  • Nama (jika diperlukan).
  • Nomor WhatsApp.
  • ID Game atau akun yang digunakan untuk pemrosesan pesanan.

2. Penggunaan Data

  • Memproses pesanan pelanggan.
  • Menghubungi pelanggan jika diperlukan.
  • Meningkatkan kualitas layanan.

3. Keamanan Data

Data pelanggan tidak akan diperjualbelikan, dibagikan, atau disebarluaskan kepada pihak lain tanpa izin, kecuali jika diwajibkan oleh hukum.

4. Cookie

Website dapat menggunakan cookie atau penyimpanan lokal (Local Storage) untuk menyimpan pengaturan seperti tema Dark/Light demi kenyamanan pengguna.

5. Persetujuan

Dengan menggunakan layanan ALL STORE, Anda dianggap telah menyetujui Kebijakan Privasi ini.
